# React@17.0.1 & React Script@4.0.1

# Typescript 4.0.3

# ESLint Prettier Config for BearSoftware.

## Usage

https://www.npmjs.com/package/@bearsoftware/ui-eslint-config

## yarn

```sh
yarn add -D @typescript-eslint/eslint-plugin@4.4.1
yarn add -D eslint@7.16.0
yarn add -D eslint-config-airbnb@18.2.1
yarn add -D eslint-config-airbnb-typescript@12.0.0
yarn add -D eslint-config-prettier@7.1.0
yarn add -D eslint-plugin-import@2.22.0
yarn add -D eslint-plugin-jest@^24.1.3
yarn add -D eslint-plugin-jsx-a11y@6.3.1
yarn add -D eslint-plugin-prettier@3.3.0
yarn add -D eslint-plugin-react@7.20.3
yarn add -D eslint-plugin-react-hooks@4.0.8
yarn add -D jest@26.6.0
yarn add -D prettier@2.2.1
yarn add -D @bearsoftware/ui-eslint-config@1.0.0
```

or
